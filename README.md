# Retail Sales Analysis Using SQL

## Project Overview

This project analyzes retail sales data using MySQL to uncover customer behavior, sales trends, product performance, and business insights.

The project includes data cleaning, data validation, exploratory analysis, and business-focused SQL queries commonly used by data analysts.

---

## Dataset

The dataset contains retail transaction information including:

* Transaction ID
* Sale Date & Time
* Customer ID
* Gender
* Age
* Product Category
* Quantity Sold
* Price Per Unit
* Cost of Goods Sold (COGS)
* Total Sale Amount

---

## Data Cleaning

Before performing analysis, the dataset was cleaned by:

* Identifying missing values
* Converting invalid zero values to NULL
* Removing incomplete records
* Validating transaction data
* Checking for duplicate and missing entries

---

## SQL Skills Demonstrated

* SELECT Statements
* Filtering with WHERE
* Aggregate Functions (SUM, AVG, COUNT)
* GROUP BY and ORDER BY
* Common Table Expressions (CTEs)
* Date Functions
* Data Cleaning Techniques
* Business Reporting Queries
* Customer Segmentation Analysis
